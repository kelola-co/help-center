## สิ่งที่คุณจะได้เรียนรู้

เมื่อจบคู่มือนี้ คุณจะสามารถ:
- ตั้งค่าแจ้งเตือนสต็อกต่ำสำหรับสินค้า
- กำหนดค่าการแจ้งเตือน
- ตอบสนองต่อการแจ้งเตือนสต็อก
- ป้องกันการขาดสต็อก
- จัดการกระบวนการสั่งซื้อใหม่

---

## แจ้งเตือนสต็อกคืออะไร?

**แจ้งเตือนสต็อก** จะแจ้งเตือนคุณเมื่อสินค้าคงคลังต่ำกว่าขีดจำกัดที่กำหนด สิ่งนี้ช่วยคุณได้:
- 🔔 ไม่ให้ขาดสินค้ายอดนิยม
- 📦 สั่งซื้อสินค้าใหม่ทันเวลา
- 💰 หลีกเลี่ยงการสูญเสียการขายจากการขาดสต็อก
- 📊 วางแผนการซื้อได้ดีขึ้น

---

## การตั้งค่าแจ้งเตือนสต็อก

### วิธีที่ 1: ต่อสินค้า

ตั้งค่าแจ้งเตือนขณะเพิ่มหรือแก้ไขสินค้า:

1. ไปที่ **สินค้า**
2. เลือกสินค้า
3. แตะ **"แก้ไข"**
4. ค้นหาฟิลด์ **"แจ้งเตือนสต็อกต่ำ"**
5. ใส่จำนวนขั้นต่ำ:
   - **สำหรับสินค้าขายเร็ว**: ขีดจำกัดสูงกว่า (เช่น 50 ชิ้น)
   - **สำหรับสินค้าขายช้า**: ขีดจำกัดต่ำกว่า (เช่น 5 ชิ้น)
   - **สำหรับสินค้าสำคัญ**: ระดับสต็อกนิรภัย (เช่น 100 ชิ้น)
6. แตะ **"บันทึก"**

### วิธีที่ 2: ตั้งค่าจำนวนมาก

ตั้งค่าแจ้งเตือนสำหรับหลายสินค้า:

1. ไปที่ **แจ้งเตือนสต็อก** (จากเมนูหลัก)
2. ดูสินค้าทั้งหมด
3. แตะแถวสินค้าใดก็ได้
4. ใส่ขีดจำกัดการแจ้งเตือน
5. บันทึก
6. ดำเนินการต่อกับสินค้าถัดไป

### วิธีที่ 3: ขณะรับสต็อก

ตั้งค่าอย่างรวดเร็วขณะรับสินค้า:

1. หลังจากรับสต็อก หน้าสินค้าจะแสดง
2. แตะ **"ตั้งค่าแจ้งเตือน"**
3. ใส่ขีดจำกัดตาม:
   - สินค้าขายเร็วแค่ไหน
   - ระยะเวลาจัดส่งของผู้ผลิต
   - ความจุของคลัง
4. บันทึก

## การเลือกขีดจำกัดการแจ้งเตือน

### ปัจจัยที่ต้องพิจารณา

| ปัจจัย | ผลกระทบต่อขีดจำกัด |
|--------|---------------------|
| **ความเร็วในการขาย** | สินค้าขายเร็วต้องการการแจ้งเตือนสูงกว่า |
| **ระยะเวลาจัดส่ง** | การจัดส่งนานกว่า = การแจ้งเตือนสูงกว่า |
| **ฤดูกาล** | เพิ่มก่อนช่วงพีค |
| **ต้นทุนคลัง** | พื้นที่จำกัด = ขีดจำกัดต่ำกว่า |
| **ความสำคัญของสินค้า** | สินค้าสำคัญ = สต็อกนิรภัยสูงกว่า |

### สูตรที่แนะนำ

**การคำนวณพื้นฐาน:**
```
ขีดจำกัดการแจ้งเตือน = (ยอดขายรายวัน × วันจัดส่ง) + สต็อกนิรภัย

ตัวอย่าง:
- ขาย 10 ชิ้น/วัน
- 7 วันจัดส่ง
- 20 ชิ้นสต็อกนิรภัย
- แจ้งเตือนที่: (10 × 7) + 20 = 90 ชิ้น
```

**แนวทางอนุรักษ์นิยม:**
```
ขีดจำกัดการแจ้งเตือน = (ยอดขายรายวัน × ระยะเวลาจัดส่ง × 1.5) + สต็อกนิรภัย
```

## การดูแจ้งเตือนสต็อก

### แดชบอร์ดแจ้งเตือนสต็อก

เข้าถึงจากเมนูหลัก:

1. แตะ **"แจ้งเตือนสต็อก"**
2. ดูสองส่วน:
   - **🔴 ต่ำกว่าขีดจำกัด** — ต้องการความสนใจ
   - **✅ เหนือขีดจำกัด** — โอเค

### ฟีเจอร์รายการแจ้งเตือน

แต่ละแถวแสดง:
| คอลัมน์ | ข้อมูล |
|---------|--------|
| สินค้า | ชื่อและรูปภาพ |
| สต็อกปัจจุบัน | จำนวนจริง |
| แจ้งเตือนที่ | ขีดจำกัดที่ตั้งไว้ |
| สถานะ | ความเร่งด่วน |

---

## การตั้งค่าการแจ้งเตือน

### การแจ้งเตือนในแอป

เปิดใช้งานเสมอ:
- จำนวนบนไอคอน
- แจ้งเตือนบนแดชบอร์ด
- แบนเนอร์ในแอป

### การแจ้งเตือน Push

กำหนดค่าที่ **บัญชี → การแจ้งเตือน**:

1. ไปที่การตั้งค่าบัญชี
2. แตะ **"การแจ้งเตือน"**
3. สลับ **"แจ้งเตือนสต็อก"**
4. ตั้งค่าระดับความเร่งด่วน:
   - **ทุกการแจ้งเตือน** — ทุกครั้งที่ขีดจำกัดถูกละเมิด
   - **เฉพาะวิกฤต** — สินค้าเร่งด่วนที่สุด
   - **สรุปรายวัน** — สรุปครั้งเดียวต่อวัน

---

## การตอบสนองต่อการแจ้งเตือน

### เมื่อคุณได้รับการแจ้งเตือน

**การดำเนินการทันที:**

1. **ตรวจสอบสต็อกปัจจุบัน**
   - ยืนยันจำนวนจริง
   - ตรวจสอบคำสั่งซื้อที่ค้างอยู่
   - ยืนยันว่าแจ้งเตือนถูกต้อง

2. **ประเมินความเร่งด่วน**
   - สินค้าขายเร็วแค่ไหน?
   - ใช้เวลานานแค่ไหนในการเติมเต็ม?
   - ลูกค้ารอได้ไหม?

3. **ดำเนินการ**
   - สั่งซื้อจากผู้ผลิต
   - โอนจากสถานที่อื่น
   - อัปเดตขีดจำกัดถ้าผิด

### การสร้างใบสั่งซื้อ

จากแจ้งเตือนสต็อก:

1. ค้นหาสินค้าในรายการแจ้งเตือน
2. แตะสินค้า
3. ดูสต็อกปัจจุบันและขีดจำกัด
4. แตะ **"สั่งซื้อสต็อก"** (ถ้ามีการผสานรวม)
5. หรือจดจำนวนที่ต้องการ
6. ติดต่อผู้ผลิต

## แนวปฏิบัติที่ดีที่สุด

### การตั้งค่าขีดจำกัด

✅ **ทบทวนเป็นประจำ**
- ทบทวนขีดจำกัดรายเดือน
- ปรับตามฤดูกาล
- อัปเดตตามแนวโน้มการขาย

✅ **แยกแยะสินค้า**
- สินค้า A (20% บนสุด): สต็อกนิรภัยสูงกว่า
- สินค้า B (30% กลาง): แจ้งเตือนปานกลาง
- สินค้า C (50% ล่างสุด): ลำดับความสำคัญต่ำกว่า

✅ **พิจารณาระยะเวลาจัดส่ง**
- ผู้ผลิตในพื้นที่: ขีดจำกัดต่ำกว่า
- ต่างประเทศ: ขีดจำกัดสูงกว่า
- ผู้ผลิตไม่น่าเชื่อถือ: เพิ่ม buffer

## การแก้ไขปัญหา

### "ไม่ได้รับการแจ้งเตือน"

**ตรวจสอบ:**
- เปิดใช้งานการแจ้งเตือนในการตั้งค่า
- ตั้งค่าขีดจำกัดการแจ้งเตือนจริงๆ
- สต็อกสินค้าต่ำกว่าขีดจำกัด
- อนุญาตการแจ้งเตือนแอป (การตั้งค่าโทรศัพท์)

**ลอง:**
- ปิดและเปิดการแจ้งเตือน
- รีสตาร์ทแอป
- ตรวจสอบการตั้งค่าการแจ้งเตือนโทรศัพท์

### "มีการแจ้งเตือนมากเกินไป"

**วิธีแก้:**
- เพิ่มขีดจำกัดสำหรับสินค้าไม่สำคัญ
- ใช้สรุปรายวันแทนการแจ้งทันที
- โฟกัสการแป้งเตือนเฉพาะสินค้า A
- ปิดการแจ้งเตือนสำหรับสินค้าขายช้า

## คำถามที่พบบ่อย

**ถาม: สามารถมีกี่สินค้าที่มีการแจ้งเตือนได้?**
ตอบ: ไม่จำกัด! ตั้งค่าการแจ้งเตือนสำหรับสินค้าได้มากเท่าที่คุณต้องการ

**ถาม: สามารถตั้งค่าการแจ้งเตือนที่แตกต่างกันสำหรับสถานที่ต่างๆ ได้ไหม?**
ตอบ: ได้ แต่ละสถานที่ธุรกิจมีการตั้งค่าการแจ้งเตือนอิสระ

**ถาม: การแจ้งเตือนทำงานแบบออฟไลน์ไหม?**
ตอบ: การแจ้งเตือนคำนวณจากข้อมูลที่ซิงค์ล่าสุด ซิงค์เมื่อออนไลน์เพื่อความถูกต้อง
