## สิ่งที่คุณจะได้เรียนรู้

เมื่อจบคู่มือนี้ คุณจะสามารถ:
- สแกนบาร์โค้ดด้วยอุปกรณ์มือถือของคุณ
- ใช้การสแกนบาร์โค้ดสำหรับการขาย
- สแกนสำหรับการดำเนินการสต็อก
- แก้ไขปัญหาการสแกน
- ตั้งค่าขั้นตอนการทำงานบาร์โค้ด

---

## ภาพรวมการสแกนบาร์โค้ด

**สิ่งที่คุณทำได้:**
- 📱 **สแกนมือถือ** — ใช้กล้องโทรศัพท์
- 🔍 **ค้นหาสินค้า** — ค้นหาสินค้าในทันที
- ⚡ **การขายรวดเร็ว** — เช็คเอาท์รวดเร็ว
- 📦 **การจัดการสินค้าคงคลัง** — การใส่สต็อกอย่างรวดเร็ว
- 🏷️ **การยืนยันสินค้า** — ยืนยันสินค้าที่ถูกต้อง

---

## ประเภทบาร์โค้ดที่รองรับ

Kelola รองรับรูปแบบบาร์โค้ดทั่วไป:

| ประเภท | รูปแบบ | กรณีใช้งาน |
|--------|--------|------------|
| **EAN-13** | 13 หลัก | สินค้าปลีกทั่วโลก |
| **UPC-A** | 12 หลัก | สินค้าปลีกอเมริกาเหนือ |
| **Code 128** | ตัวอักษรและตัวเลข | รหัสภายใน, การจัดส่ง |
| **Code 39** | ตัวอักษรและตัวเลข | อุตสาหกรรม, การป้องกัน |
| **QR Code** | 2D | URL, ข้อมูล |

## การสแกนในแอปมือถือ

### การใช้กล้องสแกน

**สำหรับการขายและสต็อก:**

1. เปิด **Stock Out** (หรือ Stock In)
2. แตะไอคอน **"สแกน"** (กล้อง)
3. ชี้กล้องไปที่บาร์โค้ด
4. ถือจนกระทั่งมีเสียงบี๊บ
5. สินค้าถูกเพิ่มอัตโนมัติในรายการ

**เคล็ดลับความสำเร็จ:**
- ✅ แสงที่ดี
- ✅ มือมั่นคง
- ✅ บาร์โค้ดชัดเจน (ไม่ยับ)
- ✅ ใกล้พอที่จะเติมเฟรม
- ❌ หลีกเลี่ยงแสงสะท้อน/เงา

### การสแกนหลายรายการ

**โหมดสแกนรวดเร็ว:**

1. รักษาหน้าจอสแกนไว้
2. สแกนรายการแรก
3. สแกนรายการถัดไปทันที
4. รายการสะสมในรายการ
5. แตะ "เสร็จสิ้น" เมื่อเสร็จ

> **เคล็ดลับโปร:** ฝึกจังหวะ — สแกน, เคลื่อน, สแกน, เคลื่อน

### การใส่ด้วยตนเอง

**เมื่อการสแกนล้มเหลว:**

1. แตะ **"ใส่ด้วยตนเอง"**
2. พิมพ์หมายเลขบาร์โค้ด
3. แตะ **"ค้นหา"**
4. เลือกสินค้า

---

## การสแกนเว็บ (USB)

### การตั้งค่า

**สำหรับเดสก์ท็อป/แล็ปท็อป:**

1. เชื่อมต่อเครื่องสแกนบาร์โค้ด USB
2. รอให้ติดตั้งไดรเวอร์
3. เครื่องสแกนทำหน้าที่เป็นอินพุตแป้นพิมพ์
4. ไม่ต้องใช้ซอฟต์แวร์เพิ่มเติม

### การใช้เครื่องสแกนเว็บ

**บน web.kelola.co:**

1. เปิด Stock In/Out
2. คลิกปุ่ม **"สแกนบาร์โค้ด"**
3. เครื่องสแกนมีเสียงบี๊บ สินค้าปรากฏ
4. จำนวนเริ่มต้นที่ 1
5. ปรับถ้าต้องการ
6. สแกนรายการถัดไป

### การกำหนดค่าเครื่องสแกน

**เครื่องสแกนส่วนใหญ่ใช้งานได้ทันที** ถ้ามีปัญหา:

1. ตรวจสอบให้เครื่องสแกนอยู่ในโหมด "HID keyboard"
2. ตรวจสอบการตั้งค่าส่งท้าย (ควรส่ง Enter)
3. ยืนยันประเภทบาร์โค้ดเปิดใช้งาน
4. ทดสอบในโปรแกรมแก้ไขข้อความก่อน

## ขั้นตอนการทำงานการสแกน

### เช็คเอาท์รวดเร็ว (ร้านค้าปลีก)

**สถานการณ์:** ลูกค้าที่มีหลายรายการ

```
1. เปิด Stock Out
2. แตะ สแกน
3. สแกนแต่ละรายการอย่างรวดเร็ว
4. ทบทวนรายการ
5. เลือกลูกค้า (ทางเลือก)
6. ส่งการขาย
7. พิมพ์ใบเสร็จ
```

**เวลาที่ประหยัด:** 50-70% เทียบกับการค้นหาด้วยตนเอง

### การรับสินค้า

**สถานการณ์:** การจัดส่งขนาดใหญ่จากผู้ผลิต

```
1. เปิด Stock In
2. แตะ สแกน
3. สแกนแต่ละรายการขณะขนถ่าย
4. ยืนยันจำนวนตามใบส่งของ
5. ส่ง stock in
6. จัดเรียงบนชั้น
```

**ประโยชน์:**
- อัปเดตสินค้าคงคลังแบบเรียลไทม์
- ไม่มีงานสะสมเอกสาร
- ตรวจพบความไม่สอดคล้องทันที

## แนวปฏิบัติที่ดีที่สุดสำหรับบาร์โค้ด

### การตั้งค่าสินค้า

✅ **กำหนดบาร์โค้ดเฉพาะ**
- หนึ่งบาร์โค้ดต่อ SKU
- ไม่ซ้ำกัน
- รูปแบบสม่ำเสมอ

✅ **พิมพ์ฉลากคุณภาพ**
- ชัดเจน, ความเปรียบต่างสูง
- ขนาดที่เหมาะสม (กว้างอย่างน้อย 1.5 ซม.)
- วัสดุทนทาน
- การวางตำแหน่งที่เหมาะสม

✅ **ทดสอบก่อนใช้งาน**
- สแกนบาร์โค้ดใหม่
- ยืนยันสินค้าถูกต้อง
- ตรวจสอบเครื่องสแกนทุกประเภท

### การทำงานประจำวัน

✅ **ความสะอาดเครื่องสแกน**
- เช็ดเลนส์กล้องเป็นประจำ
- เช็ดหน้าต่างเครื่องสแกน
- ตรวจสอบความเสียหาย

✅ **แสงสำคัญ**
- หลีกเลี่ยงแสงจ้าโดยตรง
- ตรวจสอบแสงเพียงพอ
- ปรับมุมถ้าต้องการ

✅ **แผนสำรอง**
- รู้กระบวนการใส่ด้วยตนเอง
- รักษารายการสินค้าเข้าถึงได้
- ฝึกอบรมพนักงานทุกคน

## การแก้ไขปัญหา

### "บาร์โค้ดไม่สแกน"

**สาเหตุทั่วไป:**

| ปัญหา | วิธีแก้ |
|-------|---------|
| แสงไม่ดี | ย้ายไปที่ที่สว่างกว่า |
| บาร์โค้ดเลือน | ถือกล้องให้มั่นคงมากขึ้น |
| บาร์โค้ดเสียหาย | ใส่ด้วยตนเอง |
| ประเภทบาร์โค้ดผิด | ตรวจสอบการรองรับรูปแบบ |
| แสงสะท้อน/แวววาว | ปรับมุม |
| ไกล/ใกล้เกินไป | หาระยะที่เหมาะสม |

### "พบสินค้าผิด"

**ตรวจสอบ:**
- บาร์โค้ดกำหนดให้สินค้าถูกต้อง
- ไม่มีบาร์โค้ดซ้ำในระบบ
- เครื่องสแกนอ่านรหัสทั้งหมด

**แก้ไข:**
1. ยืนยันบาร์โค้ดสินค้าในระบบ
2. กำหนดใหม่ถ้าผิด
3. ตรวจสอบบาร์โค้ดซ้ำ

## คำถามที่พบบ่อย

**ถาม: ต้องมีอุปกรณ์พิเศษในการสแกนไหม?**
ตอบ: ไม่! ใช้กล้องโทรศัพท์ของคุณ เครื่องสแกน USB เป็นทางเลือกสำหรับปริมาณมาก

**ถาม: สแกนได้โดยไม่มีอินเทอร์เน็ตไหม?**
ตอบ: การค้นหาสินค้าต้องใช้อินเทอร์เน็ต การสแกนบาร์โค้ดทำงานแบบออฟไลน์ถ้าสินค้าแคชไว้

**ถาม: ถ้าสินค้าไม่มีบาร์โค้ดล่ะ?**
ตอบ: สร้างและพิมพ์ฉลากบาร์โค้ดของคุณเอง หรือใช้การค้นหาด้วยตนเอง

**ถาม: สแกนหลายบาร์โค้ดพร้อมกันได้ไหม?**
ตอบ: ไม่ได้ สแกนทีละอัน แต่สามารถสแกนอย่างรวดเร็วต่อเนื่องได้
