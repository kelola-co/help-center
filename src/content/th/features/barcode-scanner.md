## ภาพรวม

ใช้เครื่องสแกนบาร์โค้ด USB กับ Kelola Web เพื่อเพิ่มสินค้าอย่างรวดเร็วในระหว่างการดำเนินงานสต็อก เครื่องสแกนทำงานเป็นอุปกรณ์ป้อนข้อมูลแบบคีย์บอร์ด - เพียงสแกนและบาร์โค้ดจะปรากฏในช่องค้นหา

## การตั้งค่า

### 1. อุปกรณ์สแกนเนอร์
- ซื้อเครื่องสแกนบาร์โค้ด USB จาก marketplace ใดก็ได้
- ตรวจสอบให้แน่ใจว่ามีการเชื่อมต่อ USB สำหรับการใช้งานแล็ปท็อป/PC
- เครื่องสแกนส่วนใหญ่ทำงานเป็นอุปกรณ์ "เสียบแล้วใช้งานได้ทันที"

### 2. การเชื่อมต่ออุปกรณ์
1. เชื่อมต่อเครื่องสแกนกับแล็ปท็อปหรือ PC ของคุณผ่าน USB
2. รอการติดตั้งไดรเวอร์อัตโนมัติ
3. เครื่องสแกนทำหน้าที่เป็นอุปกรณ์ป้อนข้อมูลแบบคีย์บอร์ด (โหมด HID)
4. ไม่ต้องใช้ซอฟต์แวร์เพิ่มเติม

## วิธีใช้งาน

### 1. เข้าใช้งาน Kelola Web
เปิด Kelola Web ผ่าน [web.kelola.co](https://web.kelola.co)

### 2. นำทางไปยังหน้าสต็อก
เปิดหน้าจัดการสต็อกที่คุณต้องการ:
- รับสินค้าเข้า
- ขายสินค้าออก
- ตรวจนับสต็อก

### 3. เริ่มสแกน
1. คลิกปุ่ม **สแกนบาร์โค้ด** หรือวางเคอร์เซอร์ในช่องค้นหา
2. ชี้เครื่องสแกนของคุณไปที่บาร์โค้ดของสินค้า
3. กดที่ตัวกระตุ้นเครื่องสแกน
4. หมายเลขบาร์โค้ดจะปรากฏในช่อง
5. สินค้าจะเติมข้อมูลอัตโนมัติหากพบในฐานข้อมูล

## การกำหนดค่าเครื่องสแกน

**เครื่องสแกนส่วนใหญ่ทำงานได้ทันที** หากคุณมีปัญหา:

1. ตรวจสอบให้แน่ใจว่าเครื่องสแกนอยู่ในโหมด "HID keyboard"
2. ตรวจสอบการตั้งค่า suffix (ควรส่ง Enter/Return หลังจากสแกน)
3. ตรวจสอบว่าเปิดใช้งานประเภทบาร์โค้ด (EAN, UPC, Code 128)
4. ทดสอบในโปรแกรมแก้ไขข้อความก่อนเพื่อยืนยันว่าพิมพ์ตัวเลข

## เคล็ดลับ

- เครื่องสแกนทำงานเหมือนคีย์บอร์ด - "พิมพ์" หมายเลขบาร์โค้ด
- ตรวจสอบให้แน่ใจว่าเคอร์เซอร์อยู่ในช่องที่ถูกต้องก่อนสแกน
- แสงสว่างที่ดีช่วยให้เครื่องสแกนอ่านบาร์โค้ดได้ดีขึ้น
- ถือเครื่องสแกนให้นิ่งขณะสแกน

## การแก้ไขปัญหา

**เครื่องสแกนไม่ทำงาน:**
- ตรวจสอบการเชื่อมต่อ USB
- ลองพอร์ต USB อื่น
- รีสตาร์ทเบราว์เซอร์
- ทดสอบเครื่องสแกนใน Notepad/TextEdit ก่อน

**ไม่พบบาร์โค้ด:**
- สินค้าอาจยังไม่มีในฐานข้อมูล
- อาจต้องเพิ่มบาร์โค้ดให้กับสินค้าก่อน
- ตรวจสอบการพิมพ์ผิดหรือตัวเลขที่หายไป
