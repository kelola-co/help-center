## ⚡ สิ่งที่คุณจะได้เรียนรู้

เมื่อจบคู่มือนี้ คุณจะสามารถ:
- เพิ่มสินค้าแรกลงใน Kelola
- บันทึกการซื้อสต็อก (รับสินค้าเข้า)
- บันทึกการขายครั้งแรก (ขายสินค้าออก)
- ตรวจสอบระดับสต็อกของคุณ

## 📋 ก่อนเริ่มต้น

ตรวจสอบให้แน่ใจว่าคุณมี:
- [ ] ดาวน์โหลดแอป Kelola (iOS หรือ Android)
- [ ] สร้างบัญชีหรือเข้าสู่ระบบ
- [ ] สร้างธุรกิจแรกของคุณ (หรือเข้าร่วมธุรกิจ)

> **เริ่มใช้ Kelola ครั้งแรกใช่ไหม?** ดาวน์โหลดแอปจาก [App Store](https://apps.apple.com) หรือ [Google Play](https://play.google.com)

## ขั้นตอนที่ 1: เพิ่มสินค้าแรกของคุณ

สินค้าเป็นรากฐานของสินค้าคงคลังของคุณ มาเพิ่มสินค้ากัน

1. **เปิดแอป Kelola** และแตะ **"สินค้า"** ที่ด้านล่าง
2. แตะปุ่ม **"+"** ที่มุมขวาบน
3. ป้อนรายละเอียดสินค้าของคุณ:
   - **ชื่อสินค้า** (จำเป็น): เช่น "เมล็ดกาแฟออร์แกนิก 250 กรัม"
   - **SKU/บาร์โค้ด** (ไม่บังคับ): รหัสสินค้าของคุณ
   - **หมวดหมู่** (ไม่บังคับ): เช่น "เครื่องดื่ม"
   - **ราคา** (จำเป็น): ราคาขายให้ลูกค้า
   - **สต็อกเริ่มต้น** (ไม่บังคับ): จำนวนปัจจุบันในคลัง
4. **แตะ "บันทึก"** — บันทึกสินค้าเข้าระบบเรียบร้อยแล้ว!

## ขั้นตอนที่ 2: บันทึกรับสินค้าเข้า (การซื้อ)

ต่อไปมาบันทึกสินค้าที่รับเข้าร้านของคุณ

1. จากเมนูหลัก แตะ **"รับสินค้าเข้า"**
2. คุณจะเห็นหน้าจอสองพาเนล:
   - **ด้านซ้าย**: แคตตาล็อกสินค้าของคุณ
   - **ด้านขวา**: รายการที่คุณเพิ่มลงในสต็อก
3. **ค้นหาสินค้าของคุณ** โดยเลื่อนหรือใช้แถบค้นหา
4. **แตะสินค้า** เพื่อเพิ่มลงในพาเนลขวา
5. **ป้อนจำนวน** ที่ได้รับ (เช่น 50 ชิ้น)
6. (ไม่บังคับ) **ตั้งราคาซื้อ** — ที่คุณจ่ายต่อชิ้น
7. (ไม่บังคับ) **เลือกซัพพลายเออร์** — ที่คุณซื้อมา
8. แตะ **"ส่ง"** ที่ด้านล่าง

## ขั้นตอนที่ 3: บันทึกการขาย (ขายสินค้าออก)

มาบันทึกการขายครั้งแรกของคุณกัน

1. จากเมนูหลัก แตะ **"ขายสินค้าออก"**
2. **ค้นหาและแตะ** สินค้าที่คุณต้องการขาย
3. **ป้อนจำนวน** ที่ขาย (เช่น 2 ชิ้น)
4. (ไม่บังคับ) **เลือกลูกค้า** — ติดตามว่าใครซื้อ
5. (ไม่บังคับ) **ตั้งราคาขาย** — ระบบจะใช้ราคาขายที่ตั้งไว้ให้อัตโนมัติ
6. (ไม่บังคับ) **เพิ่มบันทึก** — เช่น "ชำระเงินสด"
7. แตะ **"ส่ง"**

## ขั้นตอนที่ 4: ตรวจสอบแดชบอร์ดของคุณ

ดูธุรกิจของคุณโดยสรุป

1. แตะ **"หน้าหลัก"** ที่ด้านล่างของหน้าจอ
2. ดู **สรุปวันนี้** ของคุณ:
   - ยอดขายรวม
   - สินค้าที่ขาย
   - การแจ้งเตือนสต็อกต่ำ
3. เลื่อนลงเพื่อดู **ธุรกรรมล่าสุด**

## 🎉 ยินดีด้วย!

คุณเพิ่งเสร็จสิ้นเวิร์กโฟลว์หลักของ Kelola:
```
เพิ่มสินค้า → รับสินค้าเข้า → ขายสินค้าออก → ดูแดชบอร์ด
```

กระบวนการนี้ครอบคลุม 90% ของการดำเนินงานธุรกิจประจำวัน

## ❓ คำถามที่พบบ่อย

**ถ: ฉันสามารถใช้ Kelola บนหลายอุปกรณ์ได้หรือไม่?**
ต: ได้! ข้อมูลของคุณซิงค์โดยอัตโนมัติในทุกอุปกรณ์ เพียงเข้าสู่ระบบด้วยบัญชีเดียวกัน

**ถ: หากฉันทำผิดพลาดในการบันทึกสต็อกล่ะ?**
ต: ไปที่ **ประวัติ** ค้นหาธุรกรรม และแตะเพื่อแก้ไขหรือลบ (ต้องมีสิทธิ์เจ้าของ)

**ถ: ฉันต้องใช้อินเทอร์เน็ตเพื่อใช้ Kelola หรือไม่?**
ต: คุณต้องใช้อินเทอร์เน็ตเพื่อซิงค์ข้อมูล แต่ฟีเจอร์หลายอย่างทำงานออฟไลน์และซิงค์เมื่อเชื่อมต่ออีกครั้ง

**ถ: มีข้อจำกัดจำนวนสินค้าที่ฉันสามารถเพิ่มได้หรือไม่?**
ต: แผนฟรีมีข้อจำกัด ตรวจสอบการสมัครสมาชิกของคุณใน **บัญชี → การสมัครสมาชิก**
