## สิ่งที่คุณจะได้เรียนรู้

หลังจากอ่านคู่มือนี้ คุณจะรู้:
- ส่วนหลักของแอป Kelola
- วิธีนำทางระหว่างฟีเจอร์ต่างๆ
- ว่าจะหาฟังก์ชันเฉพาะได้ที่ไหน
- ปุ่มลัดแป้นพิมพ์ (เวอร์ชันเว็บ)

---

## โครงสร้างการนำทางหลัก

Kelola ใช้ **แถบนำทางด้านล่าง** สำหรับการเข้าถึงฟีเจอร์หลักอย่างรวดเร็ว:

<!-- TODO: Screenshot - Bottom navigation bar with labels -->

| ไอคอน | ส่วน | จุดประสงค์ |
|--------|------|-----------|
| 🏠 **หน้าหลัก** | แดชบอร์ด | ภาพรวมกิจกรรมวันนี้ |
| 📦 **สินค้า** | สินค้าคงคลัง | จัดการสินค้าและดูสต็อก |
| ➕ **เพิ่ม** | การดำเนินการด่วน | เข้าถึงงานทั่วไปได้อย่างรวดเร็ว |
| 📋 **ประวัติ** | ธุรกรรม | ดูและค้นหาบันทึกทั้งหมด |
| 👤 **บัญชี** | การตั้งค่า | โปรไฟล์ การสมัครสมาชิก ตัวเลือกอื่นๆ |

## แดชบอร์ดหน้าหลัก

ศูนย์บัญชาการธุรกิจของคุณ

### สิ่งที่คุณเห็น
- **สรุปวันนี้**: จำนวนยอดขาย รายได้ สินค้าที่ขาย
- **การแจ้งเตือนสต็อกต่ำ**: สินค้าที่ใกล้หมด
- **กิจกรรมล่าสุด**: ธุรกรรมล่าสุด
- **การดำเนินการด่วน**: ทางลัดไปยังงานทั่วไป

<!-- TODO: Screenshot - Home dashboard annotated -->

### การดำเนินการที่มีให้
- แตะที่ตัวชี้วัดใดก็ได้เพื่อดูรายละเอียด
- ดึงลงเพื่อรีเฟรชข้อมูล
- แตะที่การแจ้งเตือนเพื่อดำเนินการ

---

## ส่วนสินค้า

จัดการแคตตาล็อกสินค้าทั้งหมดของคุณ

### มุมมองรายการสินค้า
- **แถบค้นหา**: ค้นหาสินค้าทันที
- **ตัวเลือกตัวกรอง**: ตามหมวดหมู่ ระดับสต็อก สถานะ
- **ตัวเลือกเรียงลำดับ**: ชื่อ ราคา จำนวนสต็อก
- **สลับตาราง/รายการ**: เปลี่ยนโหมดการแสดงผล

<!-- TODO: Screenshot - Products list with filters -->

### การดำเนินการกับสินค้า
แตะที่สินค้าใดก็ได้เพื่อ:
- ดู/แก้ไขรายละเอียด
- ตรวจสอบประวัติธุรกรรม
- ปรับสต็อก
- แชร์ข้อมูลสินค้า

## ส่วนประวัติ

บันทึกธุรกรรมที่สมบูรณ์และการค้นหา

### ค้นหาและกรอง
- **ค้นหาตาม**: ชื่อสินค้า ลูกค้า รหัสธุรกรรม
- **กรองตาม**: ช่วงวันที่ ประเภทธุรกรรม พนักงาน
- **เรียงตาม**: วันที่ จำนวน ชื่อสินค้า

## ส่วนบัญชี

การตั้งค่าและการจัดการธุรกิจ

### รายการเมนู

#### การตั้งค่าธุรกิจ
- **สลับธุรกิจ**: เปลี่ยนระหว่างหลายธุรกิจ
- **โปรไฟล์ธุรกิจ**: แก้ไขข้อมูลธุรกิจ โลโก้ สกุลเงิน
- **การจัดการพนักงาน**: เชิญและจัดการสมาชิกในทีม
- **การสมัครสมาชิก**: ดูและอัปเกรดแผนของคุณ

#### การตั้งค่า
- **ภาษา**: เปลี่ยนภาษาแอป
- **การแจ้งเตือน**: กำหนดค่าการแจ้งเตือน
- **การพิมพ์**: ตั้งค่าเครื่องพิมพ์ใบเสร็จ
- **วิธีการชำระเงิน**: จัดการประเภทการชำระเงิน

#### การสนับสนุนและข้อมูล
- **ศูนย์ช่วยเหลือ**: เข้าถึงเอกสารนี้
- **ติดต่อเรา**: รับการสนับสนุน
- **เกี่ยวกับ**: เวอร์ชันแอปและข้อมูลทางกฎหมาย

## การนำทางเฉพาะฟีเจอร์

### หน้าการจัดการสต็อก
เมื่ออยู่ในรับ/จ่าย/ตรวจนับสต็อก:
- **พาเนลซ้าย**: แคตตาล็อกสินค้า (ค้นหาและเลือก)
- **พาเนลขวา**: รายการที่เลือก
- **แถบล่าง**: สรุปและปุ่มส่ง

<!-- TODO: Screenshot - Stock In split view -->

### หน้ารายงาน
- **ตัวเลือกวันที่**: เลือกช่วงรายงาน
- **มุมมองกราฟ**: การแสดงข้อมูลเชิงภาพ
- **มุมมองตาราง**: ตัวเลขโดยละเอียด
- **ปุ่มส่งออก**: ดาวน์โหลดเป็น PDF/Excel

## การนำทางเว็บ vs มือถือ

### แอปมือถือ (iOS/Android)
- การนำทางแท็บด้านล่าง
- ท่าทางปัดระหว่างแท็บ
- ดึงเพื่อรีเฟรช
- แตะค้างไว้สำหรับการดำเนินการด่วน

### เวอร์ชันเว็บ (web.kelola.co)
- การนำทางแถบด้านข้างซ้าย
- ปุ่มลัดแป้นพิมพ์ที่มีให้
- คำแนะนำเครื่องมือเมื่อชี้
- พื้นที่ทำงานที่กว้างขึ้นสำหรับการดำเนินการสต็อก

### ปุ่มลัดแป้นพิมพ์ (เว็บ)
| ปุ่มลัด | การดำเนินการ |
|----------|------|
| `Ctrl + K` | ค้นหาอย่างรวดเร็ว |
| `Ctrl + /` | แสดงปุ่มลัด |
| `Esc` | ปิดโมดัล/กลับ |
| `Ctrl + Enter` | ส่งแบบฟอร์ม |
| `Ctrl + 1-5` | ข้ามไปยังส่วนหลัก |

## แนวทางปฏิบัติที่ดีที่สุดในการนำทาง

### เพื่อความเร็ว
1. **ใช้แถบค้นหา** — วิธีที่เร็วที่สุดในการค้นหาสินค้า
2. **เรียนรู้ปุ่มลัดแป้นพิมพ์** (เว็บ) — ประหยัดเวลา
3. **ปักหมุดตัวกรองที่ใช้บ่อย** — เข้าถึงด้วยการแตะครั้งเดียว

### เพื่อความถูกต้อง
1. **ตรวจสอบชื่อส่วนหัว** — ยืนยันว่าคุณอยู่ที่ไหน
2. **ตรวจสอบสรุปก่อนส่ง** — ป้องกันข้อผิดพลาด
3. **ใช้ breadcrumbs** — เข้าใจเส้นทางการนำทางของคุณ

### รูปแบบการนำทางทั่วไป

```
การค้นหาสินค้า:
หน้าหลัก → สินค้า → ค้นหา → แตะสินค้า

การบันทึกการขาย:
แตะปุ่ม + → ขายสินค้าออก → เลือกสินค้า → ส่ง

การตรวจสอบรายงาน:
บัญชี → รายงาน → เลือกประเภทรายงาน → ตั้งช่วงวันที่

การเชิญพนักงาน:
บัญชี → การจัดการพนักงาน → เชิญ → ป้อนอีเมล
```

## การแก้ไขปัญหาการนำทาง

### "ฉันหาฟีเจอร์ไม่เจอ"
- ตรวจสอบ **สิทธิ์** ของคุณ — ฟีเจอร์บางอย่างต้องการการเข้าถึงเจ้าของ/ผู้ดูแลระบบ
- ดูที่ **บัญชี → เพิ่มเติม** — ตัวเลือกเพิ่มเติม
- ใช้ **การค้นหา** — ทำงานในส่วนใหญ่

### "หน้าจอดูแตกต่างออกไป"
- คุณอาจอยู่ใน **แผนต่างกัน** — ฟีเจอร์บางอย่างเป็นพรีเมียม
- ตรวจสอบ **บทบาท** ของคุณ — พนักงานเห็นอินเทอร์เฟซที่ง่ายขึ้น
- อัปเดตแอป — คุณอาจใช้เวอร์ชันเก่า

### "การนำทางช้า"
- ตรวจสอบ **การเชื่อมต่ออินเทอร์เน็ต** ของคุณ
- ปิดและเปิดแอปใหม่
- ล้างแคชใน บัญชี → การตั้งค่า → ล้างแคช
