## Membuat Barcode

### Menggunakan Aplikasi Kelola Mobile

Kelola mendukung pencetakan label barcode via printer thermal Bluetooth:

1. Buka **Lainnya** → **Cetak Barcode**
2. Pilih produk yang ingin dicetak labelnya
3. Pilih template label (58mm atau 80mm)
4. Pilih printer thermal Bluetooth Anda
5. Cetak label

**Persyaratan:**
- Printer thermal Bluetooth (kompatibel ESC/POS)
- Kertas thermal 58mm atau 80mm
- Printer dipasangkan dengan perangkat Anda

**Catatan:** Pencetakan label dilakukan melalui aplikasi mobile, bukan versi web.

### Menggunakan Alat Pihak Ketiga

**Generator Online:**
- Generator barcode gratis
- Masukkan kode, unduh gambar
- Cetak di label standar

**Opsi Software:**
- Microsoft Word (template label)
- Adobe Illustrator
- Canva
- Software label khusus

## Pengaturan Printer Label

### Jenis Printer Label

**Printer Thermal Desktop:**
- Biaya: $100-300
- Cocok untuk: Bisnis kecil
- Label: Hingga 4 inci lebar

**Printer Industrial:**
- Biaya: $500-2000+
- Cocok untuk: Volume tinggi
- Label: Format lebih besar

**Printer Standar + Label:**
- Gunakan laser/inkjet biasa
- Beli lembar label (A4)
- Cetak beberapa per lembar

### Rekomendasi Printer

**Merek Populer:**
- Zebra

- Rollo
- TSC

**Fitur Utama:**
- Pencetakan thermal (tanpa tinta)
- Konektivitas USB/Bluetooth
- Berbagai ukuran label
- Mekanisme feed yang andal

## Praktik Terbaik Desain Label

### Penempatan Barcode

✅ **Ruang kosong di sekitar barcode**
- Margin minimum 5mm
- Tidak ada teks yang tumpang tindih
- Permukaan bersih, tidak berkerut

✅ **Ukuran yang tepat**
- Tidak terlalu kecil (sulit dipindai)
- Tidak terlalu besar (membuang ruang)
- Tes pindai sebelum cetak massal

### Teks dan Font

**Nama Produk:**
- Font jelas, terbaca
- Cukup besar untuk dibaca
- Disingkat jika diperlukan

**SKU:**
- Di bawah atau di samping barcode
- Font family yang sama
- Penempatan konsisten

**Harga:**
- Menonjol tapi tidak dominan
- Simbol mata uang disertakan
- Mudah diperbarui

### Bahan Label

**Label Kertas:**
- Opsi termurah
- Baik untuk lingkungan kering
- Mungkin mengelupas di kelembaban

**Sintetis/Polyester:**
- Tahan air
- Tahan lama
- Lebih baik untuk retail

**Direct Thermal:**
- Tidak perlu ribbon
- Pudar seiring waktu
- Baik untuk jangka pendek

**Thermal Transfer:**
- Memerlukan ribbon
- Lebih tahan lama
- Lebih baik untuk label permanen

## Proses Pencetakan

### Langkah demi Langkah

1. **Siapkan Daftar Produk**
   - Ekspor dari Kelola
   - Atau buat manual
   - Sertakan: Nama, SKU, Barcode, Harga

2. **Desain Template**
   - Pilih ukuran label
   - Desain layout
   - Tes dengan satu label

3. **Hasilkan Barcode**
   - Gunakan font barcode atau generator
   - Pastikan format benar (EAN, Code 128)
   - Verifikasi dapat dipindai

4. **Cetak Halaman Tes**
   - Cetak satu lembar
   - Periksa keselarasan
   - Tes pemindaian barcode
   - Sesuaikan jika diperlukan

5. **Cetak Massal**
   - Muat kertas label
   - Cetak batch
   - Periksa kualitas

6. **Tempelkan Label**
   - Bersihkan permukaan produk
   - Tempel lurus
   - Tekan dengan kuat
   - Hindari gelembung

## Masalah Umum dan Solusi

### Barcode Tidak Bisa Dipindai

**Periksa:**
- Kualitas cetak (tidak pudar)
- Ukuran (tidak terlalu kecil)
- Kontras (gelap di terang)
- Tidak ada noda atau kerusakan

**Perbaiki:**
- Tingkatkan kegelapan cetak
- Ukuran barcode lebih besar
- Resolusi printer lebih baik
- Bersihkan lensa scanner

### Label Tidak Menempel

**Periksa:**
- Permukaan bersih
- Tidak terlalu dingin/panas
- Tipe perekat yang tepat

**Perbaiki:**
- Bersihkan permukaan sebelum menempel
- Gunakan perekat yang sesuai
- Pertimbangkan lingkungan

### Printer Macet

**Periksa:**
- Gulungan label dimuat dengan benar
- Pengaturan tegangan
- Panduan kertas sejajar

**Perbaiki:**
- Muat ulang label
- Sesuaikan pengaturan
- Bersihkan roller

## FAQ

**T: Bisakah saya mencetak barcode dari aplikasi Kelola mobile?**
J: Ya! Aplikasi mobile mendukung pencetakan label barcode via printer thermal Bluetooth. Buka Lainnya → Cetak Barcode.

**T: Format barcode apa yang harus saya gunakan?**
J: Code 128 serbaguna untuk penggunaan internal. EAN-13 untuk produk retail.

**T: Apakah saya perlu software khusus?**
J: Tidak harus. Anda bisa menggunakan template Word atau generator online.

**T: Seberapa tahan lama label cetak?**
J: Label kertas bertahan berbulan-bulan. Label sintetis bertahun-tahun. Label thermal mungkin pudar.

**T: Bisakah saya menggunakan printer biasa?**
J: Ya, dengan lembar label. Printer label khusus lebih efisien untuk volume.

**T: Bagaimana jika saya membuat kesalahan pada label?**
J: Cetak ulang dan ganti. Jangan coba koreksi tulisan tangan - terlihat tidak profesional.
