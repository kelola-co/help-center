## Overview

Use a USB barcode scanner with Kelola Web to quickly add products during stock operations. The scanner works as a keyboard input device - simply scan and the barcode appears in the search field.

## Setup

### 1. Scanner Equipment
- Purchase a USB barcode scanner from any marketplace
- Ensure it has USB connection for laptop/PC use
- Most scanners work as "plug and play" devices

### 2. Device Connection
1. Connect the scanner to your laptop or PC via USB
2. Wait for automatic driver installation
3. Scanner acts as keyboard input (HID mode)
4. No additional software needed

## Usage

### 1. Access Kelola Web
Open Kelola Web through [web.kelola.co](https://web.kelola.co)

### 2. Navigate to Stock Pages
Open your desired stock management page:
- Stock In
- Stock Out  
- Stock Audit

### 3. Start Scanning
1. Click the **Scan Barcode** button or place cursor in the search field
2. Point your scanner at the item's barcode
3. Press the scanner trigger
4. The barcode number appears in the field
5. Product auto-fills if found in database

## Scanner Configuration

**Most scanners work out-of-box.** If you have issues:

1. Ensure scanner is in "HID keyboard" mode
2. Check suffix settings (should send Enter/Return after scan)
3. Verify barcode type is enabled (EAN, UPC, Code 128)
4. Test in a text editor first to confirm it types numbers

## Tips

- Scanner works like a keyboard - it "types" the barcode number
- Make sure cursor is in the correct field before scanning
- Good lighting helps scanner read barcodes better
- Hold scanner steady while scanning

## Troubleshooting

**Scanner not working:**
- Check USB connection
- Try different USB port
- Restart browser
- Test scanner in Notepad/TextEdit first

**Barcode not found:**
- Product may not exist in database
- Barcode may need to be added to product first
- Check for typos or missing digits
